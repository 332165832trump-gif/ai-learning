\# 学习笔记：文件自动分类器



\## 001 - Windows 路径反斜杠错误



\### 我遇到的问题



我在写 Python 代码的时候，直接复制了 Windows 路径：



```python

path = "C:\\Users\\16068\\桌面"

````



结果程序直接报错。



一开始我以为是路径写错了，其实不是。



\---



\### 报错本质（我自己的理解）



Python 不是把路径当“路径”，而是当“字符串”。



而字符串里面：



```text

\\ 不是普通字符，而是“转义符”

```



比如：



\* `\\n` → 换行

\* `\\t` → tab

\* `\\U` → Unicode（这里出错）



所以：



```text

C:\\Users\\...

&#x20;    ↑

这里的 \\U 被 Python 误解了

```



\---



\### 正确写法



方法 1（最推荐）：



```python

path = "C:/Users/16068/桌面"

```



方法 2：



```python

path = "C:\\\\Users\\\\16068\\\\桌面"

```



方法 3：



```python

path = r"C:\\Users\\16068\\桌面"

```



\---



\### 我以后怎么避免



以后在 Python 里写路径：



```text

直接用 /

```



\---



\### 我真正学到的东西（最重要）



```text

程序不会理解“路径”，只理解“字符串”



问题不在 Windows，而在 Python 如何解析字符串

```
