# 编程学习笔记总索引

## 我的核心学习方法

未来真正值钱的不是"比 AI 敲代码快"，而是：

> 懂业务 + 会拆问题 + 会指挥 AI + 会验证结果 + 能把流程自动化

## 学习主线

1. **Python 自动化基础** → 文件操作、路径处理、循环判断
2. **Git / GitHub 工程化** → 版本记录、分支协作、SSH 连接
3. **AI 协作开发** → ChatGPT 规划 + OpenCode 执行 + Codex 云端协作
4. **数据采集** → 网页抓取、API 调用（下一站）
5. **金融建模与量化** → 数据分析、策略回测、风险控制（远期目标）
