# AI Learning

## 这个仓库是什么

这是我用 **ChatGPT + OpenCode + Codex** 学习 Python、Git、自动化、GitHub 工程化和未来量化能力的学习仓库。

每一行代码都是我亲手写的，每一条笔记都是我亲手整理的，每一次 commit 都记录了我的学习轨迹。

## 我的目标

我不是为了"比 AI 写代码快"，而是为了学会：

- **会用 AI**：把 ChatGPT、OpenCode、Codex 当成编程教练和协作伙伴
- **能看懂代码**：不是复制粘贴，而是理解每一行的含义
- **能修改代码**：AI 写的代码不完美，我能发现并修正问题
- **能判断对错**：培养代码审查能力，知道 AI 写得对不对
- **能拆解问题**：把模糊需求变成清晰可执行的步骤
- **能沉淀知识**：把学习过程记录到 GitHub，形成可检索的知识库
- 最终用于**自动化、数据采集、金融数据分析和量化研究**

## 我的学习方法

| 角色 | 做什么 |
|------|--------|
| **ChatGPT** | 规划路线、解释概念、纠正错误、讲清本质 |
| **OpenCode** | 本地写代码、改文件、运行测试 |
| **Codex** | 在 GitHub 上创建分支、修改文件 |
| **Git** | 记录每次修改的版本，可回溯任意历史 |
| **GitHub** | 保存和展示完整的学习轨迹 |

**学习闭环**：提出问题 → ChatGPT 拆解 → OpenCode 执行 → 我运行理解 → 写笔记 → git add/commit/push → GitHub 保存
