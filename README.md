# AI Learning

## 这个仓库是什么

这是我用 **ChatGPT + OpenCode + Codex** 学习 Python、Git、自动化和未来量化能力的学习仓库。

每一行代码都是我亲手写的，每一条笔记都是我亲手整理的，每一次提交都记录了我的学习轨迹。

## 我的长期目标

我不是为了"比 AI 写代码快"，而是为了学会：

- 会用 AI：把 ChatGPT、OpenCode、Codex 当成编程教练和协作伙伴
- 能看懂代码：不只是复制粘贴，而是理解每一行的含义
- 能修改代码：AI 写的代码不完美，我能发现并修正问题
- 能判断 AI 写得对不对：培养代码审查能力
- 能把现实问题拆成程序可以执行的流程：工程思维
- 最终把这些能力用于**自动化、数据采集、金融数据分析和量化研究**

## 我的学习方法

| 工具 | 角色 | 做什么 |
|------|------|--------|
| **ChatGPT** | 教练 | 规划学习路线、解释概念、纠正错误、讲清本质 |
| **OpenCode** | 本地工程师 | 在本地写代码、改文件、运行测试 |
| **Codex** | 云端工程师 | 在 GitHub 上创建分支、修改文件、提交 PR |
| **Git** | 记录员 | 记录每次修改的版本，可回溯任意历史 |
| **GitHub** | 档案馆 | 保存和展示完整的学习轨迹 |

**学习闭环**：
提出问题 → ChatGPT 拆解 → OpenCode 执行 → 我运行理解 → 写笔记 → git add/commit/push → GitHub 保存
